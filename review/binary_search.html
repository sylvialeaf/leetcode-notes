🌳 先决策：这题用哪套模板？

① 数组有序、要找确切的值 / 插入位置 / 边界？

→ 模板一while ≤ · mid±1
找值 → 704 / 35　·　有重复找左右边界 → 34　·　旋转有序找值 → 33（先判哪半边升序）

② 要缩到单个幸存者 / 找第一个满足条件的 / 最小化（mid 可能就是答案，不能排除）？

→ 模板二while < · right=mid
旋转数组找最小 → 153

③ 数组无序，但"答案对判定单调"（最小化最大值 / 求最小的 X 使条件成立）？

→ 二分答案在值域上二分 + check()
爱吃香蕉的珂珂 → 875（同招：1011 送包裹 / 410 分割数组）

🧩 两套模板（核心心法 · 选错会死循环）

模板一 找值 / 找边界

704 / 35 / 34 / 33 · 命中就 return，或指针交叉 left>right 结束
left, right = 0, len-1   # 闭区间 [l, r]
while left <= right:      # 带等号
    mid = (left+right)//2
    if 命中:    return ...
    elif 偏小:  left  = mid + 1
    else:       right = mid - 1

模板二 缩到幸存者 / 第一个满足

153 / 875 · mid 可能是答案不能排除，缩到 left==right 返回
left, right = lo, hi
while left < right:       # 不带等号
    mid = (left+right)//2
    if 可行/满足: right = mid    # 保留 mid！
    else:         left  = mid + 1
return left
⚠️ 成对记：right = mid 必配 while left < right；mid ± 1 必配 while left <= right。配错 → 死循环。
