# 分类对照表（对齐 灵茶山艾府 分类题单）

> 灵茶分类题单总入口：https://github.com/EndlessCheng/codeforces-go/blob/master/leetcode/README.md
> 用法：决定刷哪类时，先看这张表找到对应文件夹，再去灵茶题单挑该类的题，做完放进对应文件夹。
> 文件夹名用英文（终端要敲，避免中文乱码）；题目文件名 = 题目英文名 snake_case（如 `kth_largest.py`）。

| 灵茶 12 大分类 | 本地文件夹 | 包含啥 |
|---------------|-----------|--------|
| 滑动窗口与双指针 | `sliding_window/` `two_pointers/` | 定长/不定长窗口、双序列双指针、三指针 |
| 二分算法 | `binary_search/` | 二分查找、二分答案、最小化最大值、第K小 |
| 单调栈 | `monotonic_stack/` | 单调栈、矩形面积、贡献法 |
| 网格图 | `grid_graph/` | 网格 DFS/BFS（岛屿、腐烂橘子等） |
| 位运算 | `bit_manipulation/` | 异或、拆位、状态压缩基础 |
| 图论算法 | `graph/` | DFS/BFS、拓扑排序、最短路、最小生成树、二分图 |
| 动态规划 | `dp/` | 背包、状态机、区间DP、数位DP |
| 数据结构 | `heap/` `stack/` `queue/` `prefix_sum/` `trie/` `union_find/` `hash/` `linked_list/` `design/` | 堆/栈/队列/前缀和差分/字典树/并查集/哈希/链表/设计题(LRU等) |
| 数学算法 | `math/` | 数论、组合、几何 |
| 贪心与思维 | `greedy/` | 贪心策略、构造题、脑筋急转弯 |
| 链表、二叉树与树 | `linked_list/` `tree/` | 链表操作、二叉树遍历、一般树、回溯放 `backtracking/` |
| 字符串 | `string/` | KMP、Z函数、Manacher、字符串哈希 |

> 备注：
> - `backtracking/`（回溯）、`multi_thread/`（多线程）、`array/`（暂放归不进上面类别的数组题）是额外补充，灵茶把回溯归在"链表二叉树与回溯"一节。
> - 不用一上来就铺满所有文件夹。**跟着学习路径走到哪类，就用哪个文件夹**（路径见上级 `README.md`）。
